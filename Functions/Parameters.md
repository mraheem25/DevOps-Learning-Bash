# Parameters

### Parameters allow us to pass data to functions - making funcyons more versatile and adaptable. This enables functions to perform specific tasks based on provided input.

#### There are two types of parameters: positional and special.

Positional Parameters
- Allow us to pass data to functions and access them using numbered variables ike $1 and $2

Special Parameters
- Provide additional information about the script and the arguments passed to it
  - `$#` = holds count of the no. arguments
  - `$0` = contains name of the script 
  - `$@` = passes all arguments
