# If Statements

## If statements allow you to introduce decision making logic into your scripts. They allow you to evaluate conditions and execute different code blocks based on the result of the statement.

#### Strcuture 
```bash
#!/bin/bash
if [ condition ]
then
  #execute code block if condition is true
fi
```
The following operators allow you to compare values and determine if a conditon is true or not
              
| Operator |      Meaning     |                      
| -------- | ---------------- |                
| `-eq`    | equal to         |                
| `-ne`    | not equal to     |                
| `-gt`    | greater than     |                
| -------- | ---------------- |                
| `-eq`    | equal to         |               
| `-ne`    | not equal to     |                
| `-gt`    | greater than     |                
| `-lt`    | less than        |
| `-ge`    | greater or equal |
| `-le`    | less or equal    |
